# C Project

A standard C project structure.

## Structure

- `src/`: Source files (.c)
- `include/`: Header files (.h)
- `bin/`: Compiled executables
- `build/`: Object files and temporary build artifacts
- `tests/`: Test source files
- `lib/`: External libraries
- `Makefile`: Build system configuration

## Building

To build the project, run:
```bash
make
```

## Running

To run the main executable, run:
```bash
./bin/main
```

## Cleaning

To remove build artifacts, run:
```bash
make clean
```
